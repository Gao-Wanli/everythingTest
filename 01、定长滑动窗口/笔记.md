## 定长滑动窗口

**只需要一个索引index、具体步骤为：**

1、初始化窗口
```python
curr = 0    # 窗口（形式可能是数字、字典等）
ans = 0     # 最终结果

for i in range(0, 窗口大小):
    # 更新窗口内容
# 并保存第一次结果
```
2、窗口滑动
```python
for i in range(0, len-窗口大小):
    # 1、curr去掉左边
    # 2、curr加上右边
    # 比较后更新ans
```

### 注意1-Counter

* 从collections导入

### 注意2
* 窗口一定是内部的
* 如果求边缘最小==求内部窗口最大
